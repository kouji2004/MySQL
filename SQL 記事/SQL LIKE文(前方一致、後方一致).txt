１．LIKE句とは

　LIKE句は、主に曖昧検索を行う場合に使用する

２．LIKE句を使った様々な検索方法
　２-１．前方一致検索
　　前方一致検索をする場合は、ワイルドカードを検索条件の最後に付けます。
　　例）usersテーブル内のuserカラムに入っているデータから「ミウラ○○」のようなデータを全て取得する場合
　　SELECT user FROM users WHERE user LIKE "ミウラ%";
　　　結果
　　　 ↓
　　　ミウラコウジ〇　→ミウラ〇〇()
　　　ミウラみうら〇　→ミウラ〇〇()
　　　三浦ミウラ✕　 →〇〇ミウラ()
　　　みうらミウラ✕    →〇〇ミウラ()

２-２．後方一致検索
　後方一致検索をする場合は、ワイルドカードを検索条件の最初に付けます。
　例）usersテーブル内のuserカラムに入っているデータから「○○ミウラ」のようなデータを全て取得する場合
　SELECT user FROM users WHERE user LIKE "%ミウラ";

　　結果
　　 ↓
　　三浦ミウラ〇   　 →〇〇ミウラ(後方一致)
　　みうらミウラ〇　   →〇〇ミウラ(後方一致)
　　ミウラコウジ✕　　→ミウラ〇〇(前方一致)
　　ミウラみうら✕　　→ミウラ〇〇(前方一致)
